# 🎵 JavaScript Music Player

## 📌 Overview

The JavaScript Music Player is a web-based application that allows users to play and control audio tracks directly in the browser. It demonstrates how the HTML Audio API and JavaScript can be used to manage multimedia functionality.

## 🚀 Features

* Play and pause audio
* Next and previous track controls
* Playlist selection
* Display of song title and artist
* Progress bar indicating playback status
* Volume adjustment control
* Automatic playback of the next track

## 🛠️ Technologies Used

* **HTML5** – Structure of the music player interface
* **CSS3** – Styling and layout
* **JavaScript** – Audio control and dynamic interaction
* **HTML Audio API** – Audio playback functionality

## ▶️ How to Run the Project

1. Download or clone the repository.
2. Ensure the **songs folder contains audio files**.
3. Open **index.html** in a web browser.
4. Use the controls to manage music playback.
