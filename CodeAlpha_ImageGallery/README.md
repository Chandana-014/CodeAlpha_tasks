# 📸 Image Gallery Web Application

## 📌 Overview

The Image Gallery is a responsive web application designed to display images in a visually appealing grid layout. Users can view images, interact with them through hover effects, and open them in a larger preview mode. This project demonstrates the use of modern front-end development practices using HTML, CSS, and JavaScript.

## 🚀 Features

* Responsive image grid layout
* Hover animations for better user interaction
* Lightbox view for enlarged images
* Navigation between images
* Smooth transitions and visual effects
* Mobile-friendly design

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **JavaScript** – Interactivity and image navigation
